# Cloud Computing
Express JS rest-api

## Install the dependencies
```bash
npm install
```

### Start the app in development mode
```bash
npm start
```
Migrate database schema :
* Import d-jahit-db.sql database

Prequisites :
1. Node v16
2. MySql RDBMS

Features :
* Simple Auth
* Integrated with App Engine & Cloud-sql
* CRUD (Create, Read, Update, Delete)
  - User
  - Seller
  - Product
* Simple Search
